This is the Android app for the website rsw.me.uk/blueline.

The app makes use of a number of libraries without which development would have been much harder. Further details, and the source code, are made available for interested people.

Usage

Find a method by typing its name into the search box, then tap the name to load full details and a copy of the method diagram.

Tap on tabs to select them, or swipe the screen left and right.

Settings

Access the Settings interface via the navigation drawer, or from the overflow menu in the method view.

Database

All data used to run the application is stored on your device, and the app works without an internet connection.

Database updates currently only take place when the app itself is updated. A future update may add a feature to allow updates to the database without a full application update.
